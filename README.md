# 💧 Monitor de Hidratação (CLI)

**Versão:** 1.0.0

## 🎯 O Problema
Muitas pessoas (especialmente idosos e profissionais em trabalho focado) esquecem de beber água durante o dia, o que pode causar desidratação, dores de cabeça e fadiga.

## 💡 A Solução
Uma aplicação de linha de comando (CLI) simples e direta onde o usuário registra rapidamente a ingestão de água e o sistema calcula quanto falta para a meta diária (2000ml).

## 🚀 Como executar
1. Certifique-se de ter o Python instalado.
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute o programa: `python src/main.py`

## 🧪 Como rodar testes e linting
* **Testes:** Para rodar os testes automatizados, digite no terminal: `pytest tests/`
* **Linting:** Para verificar a formatação do código, digite: `flake8 src tests`
